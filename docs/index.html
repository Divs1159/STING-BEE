STING-BEE: Vision-Language Model for X-ray Baggage Security

An AI-powered system for threat detection in X-ray imagery.

Project Overview

STING-BEE is a vision-language model designed for **X-ray baggage security inspection**. It specializes in **scene comprehension, VQA, and grounding** for detecting concealed threats in baggage.

Qualitative Results

Dataset & Research

STING-BEE utilizes an extensive dataset for model training, including annotated baggage scans for multiple threat types.
