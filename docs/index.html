STING-BEE: Vision-Language Model for X-ray Baggage Security

An AI-powered system for threat detection in X-ray imagery.

Our Contributions

  • STCray: The first X-ray baggage security dataset with 46,642 image-caption paired scans across 21 categories, created using our STING protocol.
  • STING Protocol: A strategic method to simulate real-world baggage screening, varying item positioning, orientation, and occlusion levels.
  • STING-BEE: A domain-aware vision-language AI assistant for X-ray baggage security, supporting scene comprehension, threat localization, visual grounding, and VQA.
  • Benchmarking: STCray dataset has been benchmarked using various VLMs across multi-label classification, threat localization, detection, and VQA.
